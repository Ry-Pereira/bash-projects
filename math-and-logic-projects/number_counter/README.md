# 🔢 Number Counter Program

This is a simple Bash script that counts from 0 up to a user-specified number using a loop and displays each number in the terminal with a delay.

---

## 🧱 Project Structure

```
NumberCounter/
│
├── number_counter.sh
└── README.md
```

---

## 🚀 Features

- User input for target number  
- Counts from 0 up to the selected number  
- Uses an `until` loop in Bash  
- Displays output step-by-step  
- Adds delay (`sleep`) for better visualization  
- Simple terminal-based program  

---

## 🛠️ Technologies Used

- Bash (Shell Scripting)  
- Linux / Unix terminal commands  
- `until` loop  
- User input (`read`)  
- Delay function (`sleep`)

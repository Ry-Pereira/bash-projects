# 🧮✨ Simple Calculator Program ➕➖✖️➗

This is a simple Bash calculator script that performs basic arithmetic operations using user input. The user enters two numbers and an operator, and the program outputs the result.

---

## 🧱📁 Project Structure

```
Simple_Calculator/
│
├── simple.calulator.sh 🖥️
└── README.md 📄
```

---

## 🚀 Features 🎯

- 🧾 Accepts user input for two numbers  
- ➕ Addition (+)  
- ➖ Subtraction (-)  
- ✖️ Multiplication (*)  
- ➗ Division (/)  
- ⚠️ Operator validation  
- 🖥️ Clean terminal output  
- ⏳ Uses `sleep` for pacing  

---

## 🛠️ Technologies Used 💻

- 🐚 Bash (Shell Scripting)  
- 🧮 Arithmetic expansion (`$(( ))`)  
- 🔀 Conditional statements (`if`)  
- ⌨️ User input (`read`)  
- 🧠 Linux / Unix terminal commands

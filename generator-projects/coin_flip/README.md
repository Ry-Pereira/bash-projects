# 🪙 Coin Flip Program

This is a simple Bash script that simulates a coin flip game. The user guesses either Heads (H) or Tails (T), and the program randomly generates a result to determine if the guess is correct.

---

## 🧱 Project Structure

```
CoinFlip/
│
├── coinflip.sh
└── README.md
```

---

## 🚀 Features

- Random coin flip using `$RANDOM`
- User input (H or T)
- Input validation
- Win/Loss detection system
- Interactive terminal gameplay
- Uses delays (`sleep`) for better user experience

---

## 🛠️ Technologies Used

- Bash (Shell Scripting)
- Linux / Unix terminal commands
- Random number generation (`$RANDOM`)
- Conditional statements (`if/elif/else`)
- User input handling

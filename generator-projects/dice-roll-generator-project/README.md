# 🎲 Dice Roll Program

This is a simple Bash script that simulates rolling a dice. The program creates a short animation effect using `sleep` commands and then generates a random number between 1 and 6.

---

## 🧱 Project Structure

```
DiceRoll/
│
├── dice_roll.sh
└── README.md
```

---

## 🚀 Features

- Simulates a dice roll (1–6)
- Random number generation using `$RANDOM`
- Animated “shaking dice” effect
- Step-by-step terminal output
- Simple interactive Bash script

---

## 🛠️ Technologies Used

- Bash (Shell Scripting)
- Linux / Unix terminal commands
- Random number generation (`$RANDOM`)
- Delay functions (`sleep`)

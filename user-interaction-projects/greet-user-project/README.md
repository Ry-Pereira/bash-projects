# 👋 Greet User Bash Script

This is a simple Bash script that prints a short greeting sequence in the terminal using timed delays to create a basic animation effect.

---

## 🧱 Project Structure

```
GreetUser/
│
├── greet_user.sh
└── README.md
```

---

## 🚀 Features

- Displays a step-by-step greeting message  
- Uses `echo` for terminal output  
- Uses `sleep` to add timed delays  
- Simple beginner-friendly Bash script  
- Creates a basic animated text effect  

---

## 🛠️ Technologies Used

- Bash (Shell Scripting)  
- Linux / Unix terminal commands  
- `echo` for output  
- `sleep` for timing control  

---

1. Download or clone the repository:

